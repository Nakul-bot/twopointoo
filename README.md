# Node
The build should work with any recent version of Node.
It's recommended to use node v17.5 through NVM.

```sh
nvm install 17.5
```

# Install the dependencies
```sh
npm install
```

# Run dev environment
```sh
npm run dev
```

# How to run the static version
```sh
npm run build
npm run preview
```
